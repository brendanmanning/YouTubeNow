# YouTubeNow
Watch YouTube videos with friends
## Setup
Download [nodejs](https://nodejs.org/en/) and move to YouTubeNow folder

Run this...
```
npm install --save express@4.10.2
```
And then do...
```
npm install --save socket.io
```
Finally (inside YouTubeNow directory)...
```
node server.js
```
